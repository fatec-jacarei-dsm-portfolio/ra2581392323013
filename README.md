## Sobre

Este repositório reúne o portfólio acadêmico de **Jonatas Filipe Carvalho Ferreira**, estudante de Desenvolvimento de Software Multiplataforma na FATEC Jacareí.

O conteúdo está publicado via GitHub Pages institucional e todo o site está concentrado na pasta `docs`, conforme as orientações oficiais do curso.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript vanilla
- GitHub Pages

O projeto é totalmente estático, sem frameworks, sem npm, sem backend e sem etapa de build.

## Projetos Acadêmicos

### 1DSM - Web Wizards

Projeto introdutório para aplicar a metodologia Scrum e organizar entregas em equipe.

- **Contribuição pessoal:** participação na organização das entregas e desenvolvimento das telas do projeto.
- **Tecnologias:** HTML, CSS, JavaScript, Scrum.
- [Acessar projeto](https://cguilhermec.github.io/WebWizardsPro/)

### 2DSM - Web-Wizards / Projeto Visiona

Plataforma web para simplificar o mapeamento de uso e ocupação da terra em empresas de geotecnologia.

- **Contribuição pessoal:** atuação no desenvolvimento da interface e apoio às funcionalidades da plataforma.
- **Tecnologias:** HTML, CSS, JavaScript, geotecnologia.
- [GitHub](https://github.com/filipejonatas/WebWizardsPro-Visona)

### 3DSM - RAM.GO

Sistema para monitorar ingestão diária de calorias e nutrientes, apoiando o controle alimentar e hábitos saudáveis.

- **Contribuição pessoal:** colaboração no desenvolvimento das funcionalidades e na evolução do produto por sprint.
- **Tecnologias:** aplicação web, API, banco de dados, Scrum.
- [GitHub](https://github.com/InnoCodeSolutions/ABP_3)

### 4DSM - floatData

Solução para localização de derivadores com uso combinado de IoT, aplicação mobile e integração de dados.

- **Contribuição pessoal:** participação na solução mobile/IoT e nas integrações necessárias ao rastreamento.
- **Tecnologias:** IoT, mobile, integração de dados, Scrum.
- [GitHub](https://github.com/InnoCodeSolutions/ABP_4DSM)

### 5DSM - Mentora

Plataforma inteligente de apoio acadêmico com React Native, Node.js, TypeScript e modelos de Machine Learning para prever desempenho e risco de evasão.

- **Contribuição pessoal:** atuação como Scrum Master, desenvolvimento da API que recebia dados dos modelos de Machine Learning, criação de telas no frontend mobile e apoio à infraestrutura em cloud.
- **Tecnologias:** React Native, Node.js, TypeScript, Python, Machine Learning.
- [GitHub](https://github.com/innocode-solutions/abp_5dsm)

### 6DSM - ProconBot Jacareí

Chatbot para WhatsApp com Node.js, TypeScript, RAG e PLN, orientando consumidores a partir de fluxos do PROCON e base do Código de Defesa do Consumidor.

- **Contribuição pessoal:** desenvolvimento da API, integração com o chatbot, criação de fluxos de conversa e conexão com a base de conhecimento.
- **Tecnologias:** Node.js, TypeScript, WhatsApp, RAG, PLN.
- [GitHub](https://github.com/innocode-solutions/abp_6dsm)
